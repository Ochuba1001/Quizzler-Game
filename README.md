# Quizzler-Game
A fun and interactive quiz game application that tests your knowledge across various topics.
Quizzler is a trivia quiz application that presents users with true/false or multiple-choice questions. 
The game tracks your score and provides immediate feedback on your answers, making learning both engaging and educational.

# Features

Interactive quiz interface
Score tracking system
Multiple question categories
Immediate answer feedback
Progress tracking through questions
Clean and intuitive user interface

# Prerequisites

Python 3.7 or higher
pip (Python package installer)
